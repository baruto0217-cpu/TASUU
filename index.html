集団災害対応
フレームワーク

CSCATTT（医療指揮・調整）と METHANE（初期報告）は、集団災害（MCI）対応における国際標準フレームワークです。本アプリは現場での迅速な参照と報告作成を支援します。

CSCATTT

C
Command & Control — 指揮統制
明確なリーダーシップと指揮系統の確立

集団災害対応において最初に行うべきことは、明確な指揮命令系統の確立です。誰が現場の医療指揮を取るかを明確にし、全員がその指示に従う体制を作ります。

  • 現場医療指揮官（Medical Incident Commander）を明確に指名
  • DMAT・救急隊・消防・警察との指揮系統の調整
  • 指揮所（Command Post）の設置場所の決定
  • 役割と責任の明確な割り当て
  • 定期的な状況報告の仕組みの確立
S
Safety — 安全
3Sの原則：自己・現場・傷病者

安全確保は「自己（Self）→ 現場（Scene）→ 傷病者（Survivor）」の優先順位で行います。救助者自身が二次被害を受けることで、全体の対応能力が低下します。

  • Self：個人防護具（PPE）の着用確認
  • Scene：危険物・倒壊リスク・火災・ガス漏れの確認
  • Survivor：傷病者の安全な場所への移動
  • ホットゾーン・ウォームゾーン・コールドゾーンの設定
  • 安全ルートの確保と定期的な安全評価
ホットゾーン ウォームゾーン コールドゾーン
C
Communication — 情報伝達
効果的なコミュニケーション体制の確立

METHANEによる標準化された初期報告から始まり、継続的な情報共有体制を構築します。通信手段の確保と情報の正確な伝達が重要です。

  • METHANEによる初期報告の実施
  • 無線・衛星電話等の通信手段の確保
  • ERCSなどの災害時通信プロトコルの活用
  • 病院・DMAT・消防・警察との情報共有
  • メディア対応窓口の一元化
A
Assessment — 評価
継続的な状況・資源評価

現場の状況は常に変化します。傷病者数・重症度の変動、利用可能な資源、病院の受け入れ能力などを継続的に評価し、対応を柔軟に調整します。

  • 傷病者数と重症度分布の把握・更新
  • 現場での医療資源（人員・資器材・薬品）の評価
  • 近隣病院の受け入れ可能数の確認
  • 追加支援の必要性の判断
  • 状況の悪化・好転の継続的モニタリング
T
Triage — トリアージ
START法・JumpSTARTによる重症度分類

最大多数の最大利益（Greatest Good for the Greatest Number）の原則に基づき、限られた資源を最も効果的に配分するための重症度分類を行います。

  • 赤（I）：生命危機あり、直ちに治療が必要
  • 黄（II）：重傷だが待機可能
  • 緑（III）：軽傷、独歩可能
  • 黒（0）：死亡または救命不能
  • START法（30秒/人）による迅速なトリアージ
  • 状況変化に応じた再トリアージの実施
T
Treatment — 治療
優先度に応じた医療処置の提供

トリアージ区分に従い、治療エリアを設置して優先度の高い傷病者から処置を行います。現場での処置は生命維持に必要な最低限を行い、速やかに搬送します。

  • 治療エリア（赤・黄・緑・黒）の設置
  • 止血・気道確保・ショック対応を優先
  • SALT・MARCH（出血・気道・呼吸・循環・低体温）プロトコル
  • 医薬品・資器材の効率的配分
  • DMATや専門医チームとの連携
T
Transport — 搬送
適切な医療機関への効率的搬送

傷病者を適切な医療機関へ安全かつ迅速に搬送します。各病院の受け入れ能力を把握し、負荷が特定の病院に集中しないよう分散搬送を図ります。

  • 搬送順位の決定（トリアージ区分に基づく）
  • 救急車・ドクターヘリ・DMAT機材の効率的配置
  • 搬送先病院との事前調整・受け入れ確認
  • 分散搬送による特定病院への過負荷防止
  • 搬送記録（誰が・いつ・どこへ）の管理

METHANE

M
My call sign — 報告者識別
「私は〇〇です」から始める

報告の最初に自分のコールサイン・役職・所属を明確に述べます。受信側が誰からの報告かを即座に把握できることが重要です。

例：「こちらは〇〇救急隊、田中隊長です」

E
Exact location — 正確な位置
GPSや住所で特定できる情報を

現場の正確な位置を報告します。住所、交差点名、ランドマーク、GPS座標など、受信者が地図上で特定できる情報を伝えます。

例：「場所は〇〇市〇〇町1-2-3、〇〇駅南口前の広場です」

T
Type of incident — 事案の種類
何が起きているかを簡潔に

事案の種類と規模を報告します。交通事故・火災・爆発・建物崩壊・化学物質漏洩など、対応方針に影響する情報を簡潔に伝えます。

例：「大型バスと列車の衝突事故です」

H
Hazards — 危険因子
現在と潜在的なリスク

現在存在する危険と潜在的な危険の両方を報告します。これにより、支援隊員が適切なPPEや対応機材を準備できます。

  • 火災・爆発リスク
  • 化学物質・CBRN（化学・生物・放射線・核）汚染
  • 建物崩壊・土砂崩れリスク
  • 交通による二次事故リスク
  • 「現在なし」も明示的に報告
A
Access — アクセス
最適な進入・退出ルート

支援隊員・資器材・救急車がどこから現場に入れるかを報告します。渋滞・通行止め・最適なルートを具体的に伝えます。

例：「北側の〇〇通りから進入可能、南側は通行止めです。救急車の待機場所は〇〇公園駐車場です」

N
Number of casualties — 傷病者数
重症度別の概数を報告

傷病者の総数と重症度別の概数を報告します。初期段階では概数で構いませんが、更新を続けます。

  • 赤（重症）：〇名
  • 黄（中等症）：〇名
  • 緑（軽症）：〇名
  • 黒（死亡/救命不能）：〇名
  • 要救助者（閉じ込め等）：〇名
E
Emergency services — 緊急対応機関
必要な支援と現在の対応状況

現在現場にいる機関と追加で必要な資源・機関を報告します。DMATの要請、特殊車両の必要性なども含めます。

  • 現在対応中の機関（救急・消防・警察）の状況
  • 追加要請が必要な機関・資源
  • DMATの要請の要否
  • ドクターヘリの要請の要否
  • 特殊資器材（除染・化学・重機）の必要性

START 法フロー

① 歩行できるか？

「歩ける人はこちらへ」の呼びかけ
独歩可能 → 緑

② 呼吸はあるか？（歩行不能者）

気道確保後も呼吸なし → 黒
呼吸あり → ③へ

③ 呼吸数は？

30回/分超 → 赤
正常（10〜29回）→ ④へ

④ 末梢循環・意識

橈骨動脈触知なし / CRT 2秒超 → 赤
正常かつ指示に従える → 黄

PAT法 / MARCH

🔍
PAT = 二次トリアージ（個別詳細評価）
一次トリアージ（START法）で赤・黄タグを付けた傷病者に対し、処置優先度と搬送先を決定するために行う系統的評価。
P
Perfusion — 末梢循環評価
ショックの有無を素早く判断する。外出血があれば止血を最優先。
🩸 評価項目
  • 橈骨動脈の触知：触れない = 収縮期血圧 <80mmHg 相当 → ショック疑い
  • CRT（毛細血管再充満時間）：爪床圧迫後 2秒以上で戻らない → ショック疑い
  • 皮膚色（蒼白・チアノーゼ）・冷感・湿潤の確認
  • 外出血の有無と程度（衣服の中も確認）
⚠️ ショックの判断基準
代償性ショック：CRT延長・冷汗・不安感、血圧はまだ保たれている
非代償性ショック：橈骨動脈触知不能・意識低下・血圧低下 → 即赤タグ
🚨 対応
  • 外出血 → 直接圧迫止血を最優先（止血前に他の評価を進めない）
  • 四肢の大量出血 → ターニケット（止血帯）を使用。装着時刻を記録。
  • 内出血疑い（腹部膨満・骨盤不安定）→ 静脈路確保・搬送急ぎ
  • ショック体位（下肢挙上）※頭部外傷・呼吸困難を除く
  • 保温（低体温はショックを悪化させる）
📌 記録すること
  • 橈骨動脈：あり / なし
  • CRT：___秒
  • 外出血部位・止血方法・ターニケット装着時刻
A
Airway — 気道評価
気道の開通と維持を確認。頸椎損傷の可能性を常に念頭に置く。
🔍 評価手順
  • 話せるか？ → 発話可能 = 気道開通（最も簡単な確認法）
  • 異常呼吸音の確認：喘鳴（狭窄）・いびき音（舌根沈下）・ゴロゴロ音（分泌物）
  • 胸郭の動きを視認（左右差・奇異呼吸の有無）
  • 口腔内の異物・嘔吐物・血液の確認
  • チアノーゼ（口唇・爪床）の有無
🚨 気道確保の手技
  • 用手気道確保：頭部後屈あご先挙上法（頸椎損傷疑い → 下顎挙上法のみ）
  • 異物除去：指拭い・バッグバルブマスク・吸引器
  • エアウェイ挿入：口咽頭エアウェイ（意識ある患者には使用不可）
  • 声門上器具（LMAなど）：専門資格者が実施
  • 気管挿管：医師・救急救命士（特定行為）が実施
⚠️ 頸椎損傷が疑われる場合は頭頸部を動かさず、用手で頸部固定しながら気道確保を行う
T
Treatment — 処置の優先順位
MARCHの順で生命維持に必要な最低限の処置を行い、速やかに搬送。
✅ 現場でできる処置（優先順に）
  • M — 止血：直接圧迫・ターニケット・止血剤（ヘモスタティックガーゼ）
  • A — 気道確保：用手・エアウェイ・体位（回復体位）
  • R — 換気補助：BVM換気・緊張性気胸の減圧（胸腔穿刺）
  • C — 循環確保：静脈路確保・輸液（permissive hypotension）
  • H — 保温：毛布・アルミ保温シート・濡れた衣服の除去
⛔ 現場ではやらないこと
  • 時間のかかる詳細な身体診察（現場では最低限）
  • ルート確保に時間をかけすぎる（2回失敗したら搬送）
  • 過剰輸液（内出血が増悪する可能性）
  • 脊椎固定による搬送遅延（出血性ショックが優先）
🚑 搬送先の選定基準
  • 赤：3次救急（高度救命救急センター）
  • 黄：2次救急（救命救急センター）
  • 緑：1次・2次救急（かかりつけ含む）
🔄
二次トリアージのポイント
STARTは30秒/人の一次評価。PATで詳細評価し、タグを随時更新する。
📌 一次 vs 二次の違い
一次トリアージ（START）
目標30秒/人。歩行・呼吸・循環の3項目のみで迅速分類。資源配分の大枠を決める。
二次トリアージ（PAT）
詳細評価。処置しながら再評価。搬送先・処置内容を決定する。
  • 一次で緑タグでも症状悪化 → 直ちに再評価・タグ変更
  • 一次で黒タグでも資源余裕があれば再評価を検討
  • 処置後に必ずバイタルサインを再確認
🔁 再トリアージのタイミング
  • 処置（止血・気道確保）後
  • 搬送前（状態が変わっていないか）
  • 待機中に定期的に（目安：5〜10分ごと）
  • 患者自身が「悪化した」と訴えたとき
  • 救助者が異変に気づいたとき（呼吸変化・意識変化）
📝 タグ記録の注意点
  • 変更時は旧タグを切り取り新タグを追加（積み重ね可）
  • 記録：評価者名・評価時刻・変更理由を記入
  • 薬剤投与した場合は必ずタグに記載

CBRNE 対応

⚠ CBRNE 基本原則
近づかない・触らない・吸わない
通常のMCI対応に加え、汚染拡大防止と救助者の二次汚染防止が最優先事項。専門チーム到着まで待機が原則。
C
Chemical — 化学剤
神経剤（サリン・VX）・びらん剤（マスタード）・窒息剤（塩素）・血液剤（青酸）
🔍 識別・症状
  • 神経剤：縮瞳・流涎・気管支痙攣・筋肉痙攣・意識消失（SLUDGE症状）
  • びらん剤：皮膚水疱・眼痛・遅発性（数時間後に出現）
  • 窒息剤：咳・チアノーゼ・肺水腫（塩素特有の刺激臭）
  • 血液剤：頭痛・痙攣・急速な意識消失（アーモンド臭）
  • HAZCHEM（危険物標識・オレンジパネル）を確認
🚨 対応手順
  • 専門チーム（Level A/B PPE）が対応するまで近づかない
  • 風上・高台への避難誘導（風下は汚染リスク大）
  • 除染ライン設置：乾式（衣服除去）→ 湿式（流水15分）
  • 除染前に病院へ搬送しない（病院汚染防止）
💊 治療（専門医指示下）
  • 神経剤：アトロピン（繰り返し投与）＋ PAM（プラリドキシム）
  • 青酸系血液剤：ヒドロキソコバラミン（シアノキット）
  • 窒息剤：酸素投与・PEEP換気・安静保持
B
Biological — 生物剤
炭疽菌・天然痘・ペスト・ボツリヌス毒素・ウイルス性出血熱
🔍 特徴・識別
  • 化学剤と違い潜伏期間あり（数時間〜数週間）
  • 多数者の同時・類似症状発症が手がかり
  • 感染経路：接触感染・飛沫感染・空気感染（種類による）
  • 炭疽菌：皮膚型（黒色痂皮）・肺炎型（高致死率）
  • 天然痘：発疹の分布（全身均一、四肢に多い）
🚨 対応手順
  • N95以上のマスク・ゴーグル・ガウン着用（Level C以上）
  • 患者の隔離・コホーティング（感染経路別）
  • 保健所・感染症指定医療機関へ即時報告
  • 曝露者リストの作成と追跡調査（接触者追跡）
  • ワクチン・抗生物質の予防投与（専門機関指示）
  • 廃棄物・遺体の特殊管理（バイオハザード袋）
R

RDD（ダーティボム）・放射性物質漏洩・原子炉事故

  • 衣服除去で約80%の汚染を除去できる
🚨 対応・治療
  • 甲状腺内部被曝：安定ヨウ素剤（医師指示下・曝露前後24時間以内が有効）
  • ARS（急性放射線症候群）監視：嘔吐発症時間が重症度指標（30分以内→最重症）
  • 皮膚紅斑・脱毛・出血傾向は重症ARSのサイン
  • リンパ球数の経時測定（48〜72時間後が重要）
  • 原子力規制委員会・保健所への通報義務
NE
Nuclear / Explosive — 核・爆発物
核爆発・爆発物（IED）・テロ・建物崩壊を伴う爆発
💥 爆傷（Blast Injury）の分類
  • 一次爆傷：爆風（blast wave）→ 鼓膜穿孔・肺挫傷・腸管損傷（外見上わかりにくい）
  • 二次爆傷：飛散物による刺傷・切創・眼球損傷
  • 三次爆傷：吹き飛ばされることによる鈍的外傷・骨折
  • 四次爆傷：熱傷・有毒ガス・建物崩壊による圧挫
  • 複合損傷が多い → 体表の傷に惑わされず全身を評価
🚨 対応手順
  • 二次爆発リスクに注意：現場に安易に近づかない
  • 爆発物処理班（EOD）確認後に救助活動開始
  • フラッシュバーン（熱線熱傷）・眼球損傷の評価
  • 核爆発：EMP（電磁パルス）による通信・医療機器障害に備える
  • 核爆発の場合：爆心地から直ちに避難（距離優先）
  • 警察・自衛隊・原子力規制委員会との連携必須
🏷
CBRNE 災害トリアージの修正
汚染環境下では優先順位・搬送判断の考え方が変わる
📌 通常との主な違い
  • 「救命可能性」に加え汚染状況・除染完了の有無が判断軸に加わる
  • 除染前の傷病者は医療機関への搬送が原則禁止
  • 汚染タグ（赤/黄/緑/黒 × 汚染あり/なし）を組み合わせて管理
🏷 汚染タグの運用
赤＋汚染 除染ライン最優先通過 → 除染後に治療
黄＋汚染 除染ライン → 処置エリアへ順次移動
緑＋汚染 歩行除染（自己除染）ラインへ誘導
黒＋汚染 汚染遺体管理区域で個別に汚染管理継続
⚗️ 化学剤特有の修正
  • 神経剤大量曝露 → 痙攣・無呼吸でも除染＋アトロピン投与で回復の可能性あり（黒タグを急がない）
  • 歩行可能でも縮瞳・流涎があれば「黄」以上に変更
  • 症状なしでも曝露確認者は「緑」として経過観察
  • びらん剤：受傷後数時間で症状悪化（遅発性）→ 再トリアージ必須
☢️ 放射線特有の修正（ARS）
  • 嘔吐発症時間が重症度指標（30分以内→最重症）
  • 外見正常でも高線量曝露者は「黄」として管理
  • 外傷＋放射線の複合損傷は両方を同時評価
🚿
除染ラインの設置と運用
ウォームゾーン内に設置。除染完了前の搬送禁止。
① 乾式除染（全員に最初に実施）
  • 衣服・装備の除去で約80%の汚染を除去
  • 除去した衣服はビニール袋に密封して保管
  • 皮膚の粉塵・液滴はタオルや布で拭き取る（こすらない）
  • 時間が許せば直ちに湿式除染へ移行
② 湿式除染（乾式後に実施）
  • 大量の流水で全身を洗浄（最低15分間）
  • 眼・鼻・口腔内も流水でよく洗う
  • 歩行可能者：自己除染ライン（シャワー型）
  • 歩行不能者：担架・ストレッチャー除染ライン
  • 除染廃液の流出防止措置（土嚢・シート）が必要
🏷 除染完了の確認と搬送
  • 除染完了後に「除染済タグ／リストバンド」を装着
  • タグなし傷病者の搬送・病院受け入れは原則拒否
  • 再汚染防止のため清潔毛布・タオルで体を保護してから搬送
🗺 ゾーニング（除染配置）
ホット：汚染源・救助不可エリア
ウォーム：除染ライン設置エリア
コールド：除染完了者・指揮所・救護所
